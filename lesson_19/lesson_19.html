Синтаксические правила JSON

Основные требования:
  • Ключи всегда в "двойных кавычках"
  • Строковые значения только в "двойных кавычках"
  • Не поддерживает комментарии
  • Не допускает undefined и функции
  • Запрещены висячие запятые
Допустимые типы данных:
  • String (Строки)
  • Number (Числа)
  • Boolean (true/false)
  • null
  • Array (Массивы)
  • Object (Объекты)
Пример валидного JSON: 
{
        "name": "John",
        "age": 30,
        "city": "New York",
        "isStudent": false,
        "hobbies": ["reading", "sports"]
    }

Работа с Cookies (CRUD операции)

Основные операции:
  • Create:
    document.cookie = "username=John; expires=Thu, 18 Dec 2024 12:00:00 UTC; path=/";
  • Read (все куки):
    let cookies = document.cookie;
  • Update:
    document.cookie = "username=Jane; expires=Thu, 18 Dec 2024 12:00:00 UTC; path=/";
  • Delete:
    document.cookie = "username=; expires=Thu, 01 Jan 1970 00:00:00 UTC; path=/";
Получение конкретной куки: 
function getCookie(name) {
          let matches = document.cookie.split('; ')
              .map(cookie => cookie.split('='));
          let cookie = matches.find(item => 
              item[0] === name);
          return cookie ? 
              decodeURIComponent(cookie[1]) : 
              undefined;
      }
Параметры куки:
  • path - путь доступности
  • expires - срок действия
  • secure - только HTTPS
  • samesite - защита от CSRF
  • domain - домен доступности

Промисы в JavaScript

Что такое промисы?

Промисы — это объект в JavaScript, который представляет собой результат асинхронной операции. Они позволяют работать с асинхронным кодом более удобно и избегать "колбэк-ада".

Состояния промиса:
  • Pending (ожидание) — начальное состояние
  • Fulfilled (выполнено успешно) — операция завершена успешно
  • Rejected (выполнено с ошибкой) — операция завершена с ошибкой
Использование then:

Метод then используется для обработки успешного выполнения промиса. Вы можете использовать несколько then для последовательной обработки данных.

Пример: 
fetch('https://api.example.com/data')
        .then(response => response.json())
        .then(data => console.log(data))
        .catch(error => console.error(error));
Разделение на части:

Вы можете разделить обработку промиса на несколько частей, сохраняя промежуточные результаты в переменные. Это позволяет более гибко управлять асинхронными операциями.

Пример с разделением: 
let weather = fetch(url).then(response => response.json());
    console.log('ТЕСТ, сейчас будет погода... 3, 2, 1 ПУСК!');
    weather.then(data => console.log(data));
