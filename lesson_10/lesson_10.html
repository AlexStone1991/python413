Расшифровка оценки

Методы строк

Метод / Свойство Описание Аргументы
length Свойство, возвращающее количество символов в строке. —
charAt(index) Возвращает символ по указанному индексу. index - индекс символа (число)
substring(start, end) Возвращает часть строки между двумя индексами. start - начальный индекс (число),
end - конечный индекс (число)
slice(start, end) Извлекает часть строки и возвращает её. start - начальный индекс (число),
end - конечный индекс (число, не обязательный)
split(separator) Разделяет строку на массив подстрок по заданному разделителю. separator - разделитель (строка или RegExp)
replace(searchValue, newValue) Заменяет первое вхождение searchValue на newValue. searchValue - строка или RegExp,
newValue - строка замены
toLowerCase() Преобразует строку в нижний регистр. —
toUpperCase() Преобразует строку в верхний регистр. —
trim() Удаляет пробелы в начале и конце строки. —
includes(searchString) Проверяет, содержится ли подстрока в строке. searchString - подстрока (строка)
indexOf(searchValue) Возвращает индекс первого вхождения подстроки, или -1, если подстрока не найдена. searchValue - подстрока (строка),
fromIndex - начальный индекс (число, не обязательный)
lastIndexOf(searchValue) Возвращает индекс последнего вхождения подстроки, или -1, если подстрока не найдена. searchValue - подстрока (строка),
fromIndex - начальный индекс (число, не обязательный)
startsWith(searchString) Проверяет, начинается ли строка с указанной подстроки. searchString - подстрока (строка),
position - позиция (число, не обязательный)
endsWith(searchString) Проверяет, заканчивается ли строка указанной подстрокой. searchString - подстрока (строка),
length - длина (число, не обязательный)
concat(...strings) Объединяет две или более строк. ...strings - строки для объединения (строка)
repeat(count) Возвращает строку, повторённую указанное количество раз. count - количество повторений (число)
match(regexp) Выполняет поиск совпадений с регулярным выражением и возвращает массив совпадений. regexp - регулярное выражение (RegExp)
matchAll(regexp) Возвращает итератор с совпадениями строки с регулярным выражением. regexp - регулярное выражение (RegExp)
localeCompare(compareString) Сравнивает две строки с учётом языковых настроек. compareString - строка для сравнения (строка)
