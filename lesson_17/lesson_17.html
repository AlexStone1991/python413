    HTMLCollection и NodeList

    HTMLCollection - это живая коллекция элементов, которая автоматически обновляется при изменении DOM. В отличие от него, NodeList является статичной коллекцией узлов и не обновляется автоматически при изменениях.

    Основные отличия

    • HTMLCollection содержит только элементы, тогда как NodeList может содержать любые узлы DOM.
    • HTMLCollection является живой коллекцией, в то время как NodeList статична (за исключением childNodes).
    • NodeList предоставляет больше методов для работы с элементами.
    • HTMLCollection можно получить через методы getElementsBy..., а NodeList через querySelectorAll.

    Методы получения коллекций

    Существуют различные способы получения коллекций элементов в DOM:

    HTMLCollection

    • document.getElementsByTagName('tag') - получает элементы по имени тега.
    • document.getElementsByClassName('class') - получает элементы по имени класса.
    • document.getElementsByName('name') - получает элементы по атрибуту name.
    • document.forms - получает все формы на странице.
    • document.images - получает все изображения.
    • document.links - получает все ссылки.
    • element.children - получает дочерние элементы.
    • document.childNodes - получает все дочерние узлы документа (живая коллекция).
    • parentNode.childNodes - получает все дочерние узлы элемента (живая коллекция).

    NodeList

    • document.querySelectorAll('selector') - получает элементы по CSS селектору.
    • element.querySelectorAll('selector') - получает элементы внутри указанного элемента по CSS селектору.

    Методы для поиска одиночного элемента

    • document.getElementById('id') - получает элемент по ID.
    • document.querySelector('selector') - получает первый элемент по CSS селектору.
    • element.querySelector('selector') - получает первый элемент внутри указанного элемента по CSS селектору.

    Навигация по DOM дереву

    Родственные узлы

    • element.parentNode - получает родительский узел.
    • element.parentElement - получает родительский элемент.
    • element.closest('selector') - находит ближайший родительский элемент, подходящий под CSS селектор.

    Дочерние узлы

    • element.childNodes - получает все дочерние узлы.
    • element.children - получает все дочерние элементы.
    • element.firstChild - получает первый дочерний узел.
    • element.firstElementChild - получает первый дочерний элемент.
    • element.lastChild - получает последний дочерний узел.
    • element.lastElementChild - получает последний дочерний элемент.

    Соседние элементы

    • element.previousSibling - получает предыдущий узел.
    • element.previousElementSibling - получает предыдущий элемент.
    • element.nextSibling - получает следующий узел.
    • element.nextElementSibling - получает следующий элемент.

    Проверка наличия

    • element.hasChildNodes() - проверяет наличие дочерних узлов.
    • element.contains(elem) - проверяет, содержит ли элемент внутри себя другой элемент elem.

    Разница между Node и Element

    Node - это более общее понятие, включающее в себя:

    • текстовые узлы (просто текст)
    • узлы комментариев <!-- -->
    • элементы (теги HTML)
    • документ и его фрагменты
    • пробелы и переносы строк между тегами

    Element - это только HTML теги. Например: 

    <div>Привет <b>мир</b>!</div>
    В этом примере:
    • div является элементом и узлом.
    • Текст "Привет" является текстовым узлом.
    • b является элементом и узлом.
    • Текст "мир" является текстовым узлом.
    • Текст "!" является текстовым узлом.

    Работа с элементами

    Создание элементов

    • document.createElement("tag_name") - создает HTML элемент.
    • document.createTextNode(text) - создает текстовый узел.

    Добавление элементов

    • parent.appendChild(element) - добавляет элемент в конец родителя.
    • parent.insertBefore(element, nextSibling) - вставляет элемент перед указанным соседом.
    • parent.append(...elements) - добавляет элементы в конец родителя.
    • parent.prepend(...elements) - добавляет элементы в начало родителя.
    • parent.before(...elements) - вставляет элементы перед родителем.
    • parent.after(...elements) - вставляет элементы после родителя.

    Удаление элементов

    • element.remove() - удаляет элемент.
    • parent.removeChild(element) - удаляет дочерний элемент.

    Замена элементов

    • parent.replaceChild(newElement, oldElement) - заменяет oldElement на newElement.
    • element.replaceWith(newElement) - заменяет элемент на newElement.

    Работа с классами

    • element.classList.add('class') - добавляет класс.
    • element.classList.remove('class') - удаляет класс.
    • element.classList.toggle('class') - переключает класс (добавляет, если нет, удаляет, если есть).
    • element.classList.contains('class') - проверяет наличие класса.
    • element.className - получает или устанавливает все классы строкой.

    Работа с содержимым

    • element.innerHTML - получает или устанавливает HTML-содержимое.
    • element.outerHTML - получает или устанавливает HTML-содержимое, включая сам элемент.
    • element.textContent - получает или устанавливает текстовое содержимое.
    • element.innerText - получает или устанавливает отображаемый текст (учитывает стили).

    Работа с атрибутами

    • element.setAttribute(name, value) - устанавливает атрибут.
    • element.getAttribute(name) - получает значение атрибута.
    • element.removeAttribute(name) - удаляет атрибут.
    • element.hasAttribute(name) - проверяет наличие атрибута.
    • element.attributes - получает все атрибуты.
