Знакомство с Bootstrap 5

Подключение BS5

Bootstrap 5 предоставляет множество преимуществ для веб-разработчиков:

  • Адаптивная сетка для создания отзывчивых макетов
  • Готовые компоненты пользовательского интерфейса
  • Встроенные утилиты для быстрого стилизования
  • Кроссбраузерная совместимость
  • Активное сообщество и обширная документация

Используя Bootstrap 5, вы можете значительно ускорить процесс разработки и создавать современные, отзывчивые веб-сайты.

Подключение BS5

Подключение стилей BS5 осуществляется с помощью CDN: 


      <link
      href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css"
      rel="stylesheet"
      integrity="sha384-QWTKZyjpPEjISv5WaRU9OFeRpok6YctnYmDr5pNlyT2bRjXh0JMhjY6hW+ALEwIH"
      crossorigin="anonymous"
      />

Для подключения JavaScript-файла Bootstrap 5 используйте следующий код: 


      <script
      src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"
      integrity="sha384-YvpcrYf0tY3lHB60NNkmXc5s9fDVZLESaAA55NDzOxhy9GkcIdslK1eN7N6jIeHz"
      crossorigin="anonymous"
      ></script>
