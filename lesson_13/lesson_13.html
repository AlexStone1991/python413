Работа с Объектами

Методы объектов

Методы объектов в JavaScript

Метод Описание Аргументы
Object.keys() / Object.values() / Object.entries() Возвращают массив ключей, значений или пар ключ-значение объекта Объект
Object.assign() Копирует свойства из одного или нескольких исходных объектов в целевой объект Целевой объект, исходные объекты
Object.create() Создает новый объект с указанным прототипом и свойствами Объект-прототип, объект с дескрипторами свойств (необязательно)
Object.freeze() / Object.seal() Делают объект неизменяемым или запрещают добавление/удаление свойств Объект
Object.defineProperty() / Object.defineProperties() Определяют новое или изменяют существующее свойство объекта Объект, имя свойства, дескриптор свойства (или объект с дескрипторами)
Object.getOwnPropertyDescriptor() / Object.getOwnPropertyDescriptors() Возвращают дескриптор свойства или все дескрипторы свойств объекта Объект, имя свойства (для getOwnPropertyDescriptor)
Object.is() Определяет, являются ли два значения одним и тем же Два значения для сравнения
Object.hasOwnProperty() Проверяет, содержит ли объект указанное свойство как собственное Строка с именем свойства
Object.getPrototypeOf() / Object.setPrototypeOf() Получают или устанавливают прототип объекта Объект (и новый прототип для setPrototypeOf)
Object.preventExtensions() / Object.isExtensible() Запрещают/проверяют возможность добавления новых свойств к объекту Объект
