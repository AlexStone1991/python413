Методы массивов JavaScript

Метод/Свойство Описание Аргументы
length Свойство, возвращающее количество элементов в массиве. Нет
push() Добавляет один или несколько элементов в конец массива и возвращает новую длину массива. Элементы для добавления
pop() Удаляет последний элемент из массива и возвращает его. Нет
shift() Удаляет первый элемент из массива и возвращает его. Нет
unshift() Добавляет один или несколько элементов в начало массива и возвращает новую длину массива. Элементы для добавления
concat() Объединяет два или более массивов и возвращает новый массив. Массивы для объединения
slice() Возвращает новый массив, содержащий копию части исходного массива. start (индекс начала), end (индекс конца, не включая)
splice() Изменяет содержимое массива, удаляя или заменяя существующие элементы и/или добавляя новые. start (индекс), deleteCount (количество удаляемых), items (элементы для добавления)
forEach() Выполняет заданную функцию один раз для каждого элемента массива. callback (функция), thisArg (опционально)
map() Создает новый массив с результатами вызова заданной функции для каждого элемента. callback (функция), thisArg (опционально)
filter() Создает новый массив с элементами, прошедшими тест, заданный функцией. callback (функция), thisArg (опционально)
reduce() Применяет функцию к каждому элементу массива (слева направо) и возвращает одно значение. callback (функция), initialValue (опционально)
reduceRight() Применяет функцию к каждому элементу массива (справа налево) и возвращает одно значение. callback (функция), initialValue (опционально)
find() Возвращает первое значение в массиве, которое удовлетворяет предоставленному тесту. callback (функция), thisArg (опционально)
findIndex() Возвращает индекс первого элемента в массиве, который удовлетворяет предоставленному тесту. callback (функция), thisArg (опционально)
some() Проверяет, удовлетворяет ли хотя бы один элемент массива тесту, реализуемому предоставленной функцией. callback (функция), thisArg (опционально)
every() Проверяет, удовлетворяют ли все элементы массива тесту, реализуемому предоставленной функцией. callback (функция), thisArg (опционально)
includes() Определяет, содержит ли массив определённое значение, возвращая true или false. value (значение для поиска), fromIndex (опционально)
indexOf() Возвращает первый индекс, по которому заданное значение может быть найдено в массиве. searchElement (значение), fromIndex (опционально)
lastIndexOf() Возвращает последний индекс, по которому заданное значение может быть найдено в массиве. searchElement (значение), fromIndex (опционально)
join() Объединяет все элементы массива в строку, используя указанный разделитель. separator (опционально, по умолчанию запятая)
reverse() Изменяет порядок расположения элементов массива на противоположный. Нет
sort() Сортирует элементы массива на месте и возвращает отсортированный массив. compareFunction (опционально)
flat() Создает новый массив с подмассивами, распакованными до указанной глубины. depth (опционально, по умолчанию 1)
flatMap() Сначала отображает каждый элемент массива с помощью функции, затем объединяет результаты в новый массив. callback (функция)
