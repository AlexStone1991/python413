Виды тегов. Семантические теги. Строчные и блочные теги.

Теги: Header, Nav, Main, Footer и Code

  • Header - блочный тег, в котором мы можем расположить шапку, логотип, навигационное меню сайта
  • Nav - специальный блочный тег, в котором можно разместить навигационные менюшки
  • Main - блочный тег для основного контента сайта
  • Footer - блочный тег, где размещают контактную информацию, карту, информацию о доставке и т.п. вещи
  • Code - строчный тег, для размещения коротких блоков кода (в одну строку)
  • Section - специальный тег, для разделения контента на целостные, отностельно самостоятельные части
  • Mark- тег-маркер, который позволяет выделить текст в контексте, например, для выделения важных слов в тексте.

Section - специальный тег, для разделения контента на целостные, отностельно самостоятельные части

Section имеет нюансы, напрмер должен содержать заголовок второго уровня перед открытием или сразу после него.

Div и Span

Div и span - это универсальные теги, которые используются для группировки и стилизации элементов на веб-странице.

Div - это блочный элемент, который используется для создания контейнеров или разделов на странице. Он может содержать внутри себя другие блочные и строчные элементы. Div часто применяется для структурирования и позиционирования элементов с помощью CSS.

Span - это строчный элемент, который используется для выделения и стилизации небольших фрагментов текста или других строчных элементов. Span не создает новую строку и не имеет отступов по умолчанию. Он часто применяется для изменения цвета, шрифта или других свойств отдельных частей текста.

Хотя div и span не имеют семантического значения, они очень полезны для гибкой верстки и стилизации элементов на странице. Однако, по возможности, рекомендуется использовать семантические теги, такие как header, nav, main, footer и другие, для улучшения структуры и доступности веб-страницы.

Семантическое выделение жирным и обычное. Курсив

Для выделения текста жирным шрифтом в HTML используются два тега: <b> и <strong>. Хотя визуально они выглядят одинаково, между ними есть семантическая разница.

  • <b> - это обычный тег для выделения текста жирным шрифтом без придания ему особого значения. Например: этот текст будет жирным.
  • <strong> - это семантический тег, который указывает на важность выделенного текста. Используйте его, когда хотите подчеркнуть значимость информации. Например: этот текст важный и будет жирным.

Для выделения текста курсивом в HTML также есть два тега: <i> и <em>.

  • <i> - это обычный тег для выделения текста курсивом без придания ему особого значения. Например: этот текст будет курсивом.
  • <em> - это семантический тег, который указывает на акцент или эмоциональную окраску выделенного текста. Используйте его, когда хотите передать интонацию или настроение. Например: этот текст с акцентом и будет курсивом.

Старайтесь использовать семантические теги <strong> и <em>, когда это уместно, для улучшения доступности и понимания вашего контента. Однако, если вам просто нужно визуальное выделение без семантического значения, можете применять теги <b> и <i>.

Ссылки

Ссылки в HTML создаются с помощью тега <a>. Атрибут href указывает на целевой адрес ссылки.

Форматы ссылок

  • Ссылка на веб-страницу: Пример сайта
  • Ссылка на адрес электронной почты: Написать письмо
  • Ссылка на телефонный номер: Позвонить +7 (495) 123-45-67

Создание навигационного меню

Для создания навигационного меню можно использовать ссылки с якорями на заголовки второго уровня (<h2>) на этой же странице. Для этого нужно добавить атрибут id к заголовкам и указать его значение в атрибуте href ссылки.

    
<nav>
  <ul>
    <li><a href="#tags">Теги</a></li>
    <li><a href="#div-span">Div и Span</a></li>
    <li><a href="#formatting">Форматирование</a></li>
    <li><a href="#quotes">Цитаты</a></li>
    <li><a href="#code">Блоки кода</a></li>
    <li><a href="#links">Ссылки</a></li>
  </ul>
</nav>

Не забудьте добавить соответствующие атрибуты id к заголовкам <h2> в каждой секции:

    
<section>
  <h2 id="tags">Теги</h2>
  ...
</section>

<section>  
  <h2 id="div-span">Div и Span</h2>
  ...
</section>
